# Tweeter Project 🔊

- Tweeter is a simple, single-page Twitter clone.

  # Technologies applied 🎒

  - Front End:

        HTML, CSS, JS, jQuery, AJAX

  - Back End
          
        Node, Express

## Getting Started ⤵️

1. Install dependencies using the command: 
  
        npm install
2. Start the web server using the command: 

        npm run local
                  
3. In your browser go to:

    <http://localhost:8080/>

## Dependencies 💾

    Body-parser
    Chance
    Express
    Md5
